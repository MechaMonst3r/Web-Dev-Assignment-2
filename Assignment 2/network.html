Different Network Protocols

Transmission Control Protocol
Transmission Control Protocol(TCP) uses a set of rules to exchange messages with other Internet points at the information packet level. Learn more!
User Datagram Protocol
User Datagram Protocol(UDT) acts as an alternative communication protocol to TCP. It is used to establish low-latency and loss-tolerating connections between applications and the Internet. Learn more!
Internet Protocol
Internet Protocol(IP) uses a set of rules to send and receive messages at the Internet address level; and additional protocols like Hypertext Transfer Protocol (HTTP) and File Transfer Protocol(FTP). Learn more!
